Project Details

In the Philosopher project at 42 school, I tackled the classic dining philosophers problem using multithreading and synchronization techniques. The goal was to simulate a scenario where multiple philosophers share a dining table with limited resources (forks) and must avoid deadlock and starvation while following certain rules for eating. By implementing this project, I delved into the intricacies of concurrent programming, mutexes, and semaphores. It provided a hands-on exploration of thread management and synchronization mechanisms in a multithreaded environment. This project deepened my understanding of parallel computing and the challenges associated with resource sharing and coordination among concurrent processes.
